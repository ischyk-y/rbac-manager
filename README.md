## Короткий опис
Access Control & RBAC Manager - це desktop-система на Electron + React + SQLite для керування доступом за моделлю RBAC (Role-Based Access Control).  
Застосунок дозволяє адмініструвати користувачів, ролі, групи та матрицю прав, а також вести аудит змін і експортувати дані у CSV.

## Інструкція із запуску
1. Встановити залежності:
```bash
npm install
```
2. Запустити застосунок:
```bash
npm start
```

Додатково для режиму розробки:
```bash
npm run dev
```

## Реалізовані фічі
- Управління користувачами: створення, редагування, видалення, фільтрація, пошук, сортування, пагінація.
- Управління ролями: створення/редагування ролей, пріоритет ролей.
- RBAC-матриця прав: редагування дозволів ролі для кожного ресурсу.
- Призначення ролей користувачам і перегляд ефективних прав.
- Групи та групові призначення ролей.
- Журнал аудиту з фільтрами, сортуванням і пагінацією.
- Експорт журналу аудиту в CSV через нативний системний діалог.
- Імпорт/експорт користувачів у CSV.
- Нативні системні сповіщення (Electron Notification).
- Auto-refresh з контролем інтервалу (вимкнено/5/10/30/60 сек).
- Keyboard shortcuts для навігації, оновлення і перемикання теми.
- Світла/темна тема інтерфейсу.
- Станові повідомлення UI: loading, empty, error + кнопка «Спробувати ще раз».
